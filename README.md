# Mini Visual Timer

화면 위에 항상 떠 있는 미니 비주얼 타이머 데스크탑 앱입니다. 물리 타이머(예: Time Timer)처럼 다이얼을 돌려 시간을 설정하면, 남은 시간만큼 파이가 줄어드는 방식으로 동작합니다.

## 실행 방법

```bash
npm install
npm start
```

## 사용법

- 다이얼을 클릭 & 드래그하여 시간 설정 (최대 60분)
- 하단 **시작** 버튼으로 카운트다운 시작 / 정지
- 다이얼 더블클릭 시 리셋
- 시간이 다 되면 알림음 재생
- 창 상단 바를 드래그하여 위치 이동, × 버튼으로 종료
- 항상 화면 최상단에 고정되며 다른 Space(가상 데스크탑)로 이동해도 따라옴

## 기술 스택

- Electron
- HTML / CSS / Vanilla JS (SVG 기반 다이얼 렌더링)

## 폴더 구조

```
main.js       # Electron 메인 프로세스 (창 생성/설정)
preload.js    # 렌더러 ↔ 메인 프로세스 브릿지
index.html    # 타이머 UI 마크업
style.css     # 스타일
renderer.js   # 타이머 로직 (다이얼 그리기, 카운트다운, 사운드)
```
